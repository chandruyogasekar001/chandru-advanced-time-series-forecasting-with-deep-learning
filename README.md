This project implements an **advanced time series forecasting pipeline** that goes beyond traditional statistical methods such as ARIMA and SARIMA. A core focus of this project is **model interpretability**, achieved through **Explainable AI (XAI)** using **SHAP**, making the solution suitable for regulated and high-stakes environments.
